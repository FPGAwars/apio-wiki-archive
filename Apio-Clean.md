# Usage

```bash
apio clean [OPTIONS]
```

# Description

Clean all the generated files: blif, asc, bin, rpt, vcd and out.


# Options

| Flag | Long Flag    | Description |
| ---- | ------------ | ----------- |
| `-p` | `--project-dir` | Set the target directory for the project |
